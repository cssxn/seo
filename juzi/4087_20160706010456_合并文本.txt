中新网11月17日电(鲍文玉) 距离2016央视猴年春晚还有不到三个月的时间，各类节目正在进行紧张筹备中。16日，央视猴年春晚语言类节目一审在央视老台举行，媒体透露，当日仅有4个节目参与，朱迅、赵忠祥、曹云金、刘云天、苗阜、王声，及首次参加春晚的笑星乔杉、修睿等纷纷现身，现场由尼格买提主持。</p>
值得一提的是，为了更好地对节目“笑果”进行测评，今年春晚剧组在北京各大高校精选部分大学生参与审查，而为了防止提前泄露节目内容，剧组还与同学们签了一份很严格的保密协议。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 图片来源：张杰微博截图
语言类4节目参审 传谢娜将搭档李菁演小品</p>
临近春晚，不少节目已经在筹备中。而此前冯巩、岳云鹏、郭冬临等笑星都曾亮相影视之家，16日的审查中，仅有4个节目参与。当天下午，朱迅、赵忠祥，曹云金、刘云天、苗阜、王声，以及笑星乔杉、修睿等纷纷亮相央视老台。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;苗阜王声 图片来源：大连晚报
据悉，首度参与春晚审查的乔杉、修睿带了一个原创小品节目参与审查，而苗阜与王声今年则有两个节目参加一审。苗阜表示，他们今年准备了五个节目，但目前只留下了两个：“我的两个剧本都已经通过审查了，我个人觉得文本本身很不错，现在还没进入排练，不知道作品能不能立起来。目前据我所知，语言类节目现在只有7个，剧本审查阶段就淘汰了一批。后面还会有很多次审查。”</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 曹云金 图片来源：京华时报
据《兰州晨报》报道，有消息称，今年谢娜很有可能搭档相声演员李菁出演小品类节目。15日当天谢娜已提前参与了春晚语言类节目一审。有知情人透露，与之搭档的是相声演员李菁。被问到当天是否参与了春晚小品的录制时，李菁以春晚之事不能透露为由，挂断电话。</p>
此外，日前“开心麻花”沈腾也透露，今年的春晚作品还在准备的过程中，“已经和春晚导演组开过会了，他们越来越急，我们头越来越大，春晚小品实在太难了。”针对很多“郝建迷”想问的是否会再演“郝建”系列续集，沈腾表示：“是否延续郝建这个是需要本子本身去决定的，如果本子需要郝建还会再次出现，如果不需要，就会是另外一个人物。”</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 傅琰东 来源：西宁晚报
刘谦谢绝春晚邀请 傅琰东秘密筹备作品</p>
此前，魔术师刘谦曾连续四年登上春晚舞台，在2014年退出春晚后，今年他又一次谢绝春晚邀请。不过据《华西都市报》报道，同样上过央视春晚的著名魔术师傅琰东，在央视宣布吕逸涛出任猴年春晚总导演后，已接到上春晚的邀请，他和团队一直在认真秘密准备。</p>
上周末刘谦现身深圳，采访时他表示，因为春晚太累，所以今年还是不上春晚。他说道，“其实年年都有收到邀请，但今年不会上，春晚是一件非常折磨人的事，想多休息一段时间。”今年8月，刘谦升级成为奶爸，他笑言在家从来不会表演魔术给老婆孩子看，“研究魔术是兴趣，表演是工作，但必须要和工作分开。”</p>
