张亮靠水煮牛肉追到老婆</p>
自从录制《爸爸去哪儿》第一季后，张亮不仅因为帅气的外形、谦逊的性格赢得观众喜爱，其高超的厨艺也十分抢眼。</p>
据悉，张亮15岁之前，是个每月有近3000元零花钱、衣食无忧的孩子，但之后家里发生变故。16岁，他被迫辍学打工，第一份工作就是粤菜帮厨，“我在酒店里去做杀鱼的工作做了三个月，然后慢慢晋升”。</p>
此后，他更靠厨艺追到了妻子，“第一次给我老婆做菜还是在追她的时候，那会儿做的就是水煮牛肉。我现在都记得她当初吃的表情，一下子就觉得有戏，应该是追到手啦”。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;图片来源：黄磊微博
黄磊自称“黄小厨”：何炅吃我做的饭最多</p>
与张亮相似，黄磊是在录制《爸爸去哪儿》第二季时，高超的厨艺才被大众所熟知。但在观众惊叹的同时，黄磊却称这只是小儿科，“我的父亲母亲都比较会做菜，我很小的时候都是在家吃饭，所以老是看爸妈做就有印象，我很喜欢做饭，很享受在厨房里调这些味道，而且我还特别爱N瑟，开始一帮朋友在一块，还你做一个菜，我做一个菜，后来大家都发现我做的好吃，就改成我包办了”。</p>
黄磊同时也坦言对做饭有爱好，“做饭对我来说纯粹是爱好，特快乐，我也特爱琢磨”。还称尝过他手艺的明星不少，“何炅是吃我的饭最多的，这帮朋友没事会经常到我家来聚会，做一桌子菜，还挺好玩的”。</p>
微博上，他自称“黄小厨”，经常晒出自己做的各种美食，引发关注。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;图片来源：叶一茜微博
叶一茜贴心写菜谱</p>
此前，叶一茜偶尔会在微博晒出制作菜肴的成品，例如第一次烧海参等。自参加了《星厨驾到》后，她还会配上手写菜谱，十分贴心。</p>
微博中，叶一茜不仅尝试过酱爆牛蛙、销魂土匪鸭等菜式，更能熟练制作各种汤，以及冰淇淋等。对此，她自恋表示，“我的厨艺一定是遗传了我的颜值”，而网友则纷纷称赞她上得厅堂，下得厨房。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 资料图：林依轮(左)
林依轮曾在国外当厨师</p>
今年7月，由林依轮主持的美食节目在北京举办发布会，他还现场展示了制作酸汤鱼等菜品的过程。他表示，自己很喜欢烹饪，在家也是大厨，“虽然平时不是天天在家做饭，但节假日、纪念日会做饭”。对于有网友表示在家做饭麻烦，林依轮否认道：“其实只要自己静下心来，在厨房也能有大师的感觉。”</p>
其实，林依轮的厨艺是在踏上音乐道路之后才磨练出来的，当年他到广州发展时，因为吃不惯当地饮食，就常打电话回家问某道菜怎么做，又觉得话费太贵，于是改成写信，信中夹着菜谱。后来，他出国游历，“那个时候在外，不是想做什么就能做什么，而餐饮可能是华人在外都会经历的第一步。我在当地的中餐厅很快当上了厨师，干了一年”。</p>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 图片来源：邓紫棋微博
邓紫棋爱做西式菜品</p>
性格活泼的邓紫棋，私下喜欢研究西式菜品及糕点。此前，她曾多次在微博晒出自己制作的三明治、煎蛋等食物，还尝试过制作翻糖蛋糕为姐妹庆生。不仅食物配色鲜艳美丽，且搭配也十分健康。</p>
此外，她还曾在采访中，将自己制作的饼干与媒体分享。</p>
中新网9月10日电 昨晚，TFBOYS成员王源通过微博晒出一张自己和爱犬的合照，并留言称：“论一只泰迪，是怎么变成泰迪......熊的”。</p>
照片中，王源一手拿着手机，一手抱着泰迪犬，手机壳十分抢镜。网友看到照片后纷纷留言，“这抱的姿势。好有喜感”、“手机壳也太非主流了吧”、“小狗好可爱”、“萌萌哒”。</p>
中新网9月9日电 据台湾“东森新闻”报道，经历涉毒案后，柯震东工作量锐减，久未在媒体面前曝光。今日，他出席代言活动记者会，坦言很久没演戏，最近有点技痒。</p>
谈到好友王大陆走红，柯震东称并不担心自己的名气被超越，坦言：“就像陈柏霖当年也不会担心我超过他！”他还打趣地说：“未来会有越来越多新人爆红，但是新人不要太嚣张，会像我一样跌得鼻青脸肿。”</p>
中新网9月10日电 据台湾“东森新闻”消息，近期，刘烨带爱子诺一录制《爸爸去哪儿3》，得到观众喜爱。节目中，4岁的诺一性格直率可爱，长相帅气俊俏，初登场时就成为人气王。但是，过度的关注也让他的私生活受到了影响，近日，刘烨接受采访时透露，儿子出门和妈妈购物会被大家围观拍照，并希望大家“忘了他吧”。
刘烨坦言诺一因为在节目中人气大增，出门常常被路人围观拍照，而诺一也对此产生了排斥的情绪，就连在《爸爸去哪儿3》中看见自己都不太习惯。他表示，小孩即将开始上学，很怕因为孩子人气太旺，在人际上出现阻碍，被其他小朋友用异样的眼光看待，所以等节目结束播出后，就会减少让爱子露面的机会，希望让大家忘了诺一。</p>
此外，对于经常在微博称爱子不帅，刘烨解释称背后有很深沉的原因，他不希望小孩萌生“大家都喜欢我”的想法，所以想借此加以引导，私下还会用很严厉的态度告诉宝贝，“不是所有人都喜欢你，只是爸爸让不喜欢你的人都见不到你，世界上肯定有人不喜欢你”，希望通过这样的方式，让孩子提升面对挫折和委屈的能力。
核心提示：前中央电视台主持人方宏进经营的北京澳卫时代广告传媒有限公司(以下简称澳卫公司)因与今麦郎日清食品有限公司(以下简称今麦郎公司)发生广告合同纠纷，2009年10月11日，河北省邢台市隆尧县公安局以“涉嫌合同诈骗罪”将方宏进刑事拘留，2011年9月13日隆尧县检察院以证据不足为由作出不起诉决定。
日前，方宏进向《法制晚报》记者(微信公号：fzwb_52165216)表示，此案中隆尧县公安局所扣押的120万元(100万本金，20万利息)涉案款，一直未按法律程序返还，这笔钱，他已索要了6年。</p>
涉嫌合同诈骗 方宏进遭网上追逃</p>
方宏进2003年离开中央电视台，跳槽到东方卫视担任总策划和制片人，之后投资创办了澳卫公司，主要经营影视剧广告的植入业务。</p>
“当时，有朋友在拍一部叫《候机大厅》的情景剧，澳卫公司参与了广告植入的招商，招商时结识了华龙日清食品有限公司(现今麦郎日清食品有限公司)的董事长范现国，经双方协商，在主要场景里设置‘今麦郎面馆’这一植入广告场景。”方宏进说。</p>
2006年3月双方签订了广告合同，同年4月今麦郎公司向澳卫公司支付了100万元首付款，但由于档期等原因，情景剧迟迟未能播出。</p>
“电视剧拍完后，因档期等原因，未能按合同约定的时间播放，为了补偿今麦郎公司的损失，我们双方口头约定把植入广告置换成了普通广告。从2007年5月1日起播了一个月，共计50分钟，如按照央视的公开报价，该组广告价值为252万元。”方宏进告诉法晚记者(微信公号：fzwb_52165216)。</p>
方宏进还表示，在随后两年的时间里，双方为妥善解决合同的遗留问题，多次以信函和面谈的方式进行协商。但一直未达到双方满意的结果。</p>
2008年6月份，今麦郎向隆尧县公安局报案称，方宏进等三人涉嫌合同诈骗，同年6月11日，方宏进等三人被隆尧县公安局网上追逃。</p>
2008年6月17日，河北省邢台市隆尧县公安局将在北京市海淀区的澳卫公司副总经理温东明抓获，押至了隆尧县。</p>
“知道温东明被抓的消息后，我马上委托律师联系隆尧县公安局。对方称，温东明等涉嫌合同诈骗，接到报案后隆尧县公安局于6月11日开始网上追逃，名单上包括我、温东明、于宏伟(方宏进前妻、澳卫时代法定代表人)在内三个人。”方宏进说。</p>
随后，隆尧县公安局扣押了方宏进120万元涉案款，温东明获得保释。</p>
被拘三天 检方做出不起诉决定</p>
“我本以为这事交钱后就算结了。哪知道，温东明的事完了，我的事还没完。”方宏进对法晚记者(微信公号：fzwb_52165216)表示。</p>
2009年10月4日，方宏进从深圳出关去香港，做一起基金项目，被边检人员扣押。</p>
“当时不知道什么原因把我扣押了，我还担心自己被怀疑从事洗钱交易，因为基金项目里有一笔来自中东的资金。后来隆尧公安局的人来了，把我带回隆尧。”方宏进回忆说。</p>
2009年10月11日，方宏进依然以涉嫌合同诈骗罪，被隆尧县公安局刑事拘留，三天后取保候审。</p>
方宏进告诉法晚记者：“我和律师一致认为，这纯属一起经济合同纠纷案，完全可以通过民事诉讼解决，如今被定成合同诈骗，我们不认同。”</p>
2010年10月13日，方宏进涉嫌合同诈骗一案，被隆尧警方移送检察院审查起诉。</p>
2011年9月13日，隆尧检察院对方宏进涉嫌合同诈骗一案做出不起诉决定。</p>
“因为这起案件，直接导致了公司的停业，前期投资承包电视广告的650万也没收回，身上还背负了200多万的债务。”方宏进苦笑着说。</p>
记者了解到，此案被媒体相继报道后，方宏进曾在2009年10月21日发表声明：此次事件给我的亲友和各方合作伙伴带来了巨大的困扰，对此我表示深深的歉意。我在创业过程中得到若干朋友在各方面的鼎力支持，他们至今仍然对我信任有加，对此我表示由衷的感谢，并一定会以实际行动不辜负他们的理解、支持和期待。我希望此事能够尽快依法解决，让涉及的企业和个人都能够安心地从事各自的工作。</p>
120万扣押款 要了6年未返还</p>
“隆尧县检察院做出不起诉决定之后，隆尧县公安局理应把扣押的120万元涉案款，还给我们，但至今尚未返还。”方宏进补充说。</p>
方宏进表示，隆尧县检察院作出不起诉决定之后，其和律师多次通过各种渠道联系隆尧县公安局，索要120万扣押款，但对方均未给予任何回复。</p>
据《河北省公安机关办理经济犯罪案件程序规定》第101条法规显示：对于扣押的物品、文件、邮件、电子邮件、电报，应当指定专人妥善保管，不得使用、调换、损毁或者自行处理。经查明确实与案件无关的，应当自查清之日起三日内解除扣押，退还原主或者原邮政部门、网络服务单位。</p>
方宏进还表示，因为此事自己与前妻的婚姻也宣告破裂，“她是澳卫公司的法定代表人，因为此案也被网上追逃，觉得我连累了她，对我有怨恨。”</p>
“公司停业以后，身上背负了200多万 的债务，生活非常拮据，曾经100块钱，苦撑一个月。后来案子结束之后，就一直没有去什么单位任职，除了给企业做一些咨询之外，大部分时间都在做《汉奸史》这套丛书的编写。”方宏进补充说。</p>
“我现在每年都会到河北的公检法系统去申诉，我希望不是简单还我钱，而是要调查清楚这个事件的原委。”方宏进最后说。</p>
据了解，方宏进于1986年南开大学社会学系研究生毕业之后，出任深圳大学管理系教师、系副主任，从1991年起开始从事新闻工作。1991年在北京电视台参与制作并主持《18分钟――经济与社会》栏目，1992年进入中央电视台，先后加入《观察思考》、《东方时空》栏目。</p>
1995年，方宏进参与创办并长期主持《焦点访谈》栏目，长达8年。</p>
2003年，方宏进离开中央电视台，出任东方卫视总策划、《看东方》栏目总制片等。</p>
2006年年底，方宏进离开东方卫视。随后与妻子投资创办了澳卫公司，经营影视剧广告植入等业务。</p>
一笔120万的“扣押款” 隆尧公安发还今麦郎</p>
“我已经问过当时主办此案的李队长了，他说这笔120万的款项，已经返还给今麦郎了，手续都有，随卷移交给检察院了。”隆尧县公安局负责宣传的谢警官在接受法晚采访时说。</p>
随后，法晚记者采访了隆尧县检察院，政治处的乔银海表示：“隆尧县公安局没有往这移交，我们也没有见到这笔扣押的款项。”</p>
另外，方宏进的代理律师莫少平向记者提供了一份隆尧县公安局发出的“扣押物品、文件清单”。清单备注一栏写明：“北京莫少平律师事务所代澳卫时代广告传媒有限公司退还河北华龙日清食品有限公司广告预付款(赃款)100万元及赔付利息款20万元。”落款时间为：2008年7月18日。</p>
在另一份随卷的隆尧县公安局的“发还扣押物品、文件清单”中，内容为：人民币120万元。备注一栏写明：发还今麦郎食品有限公司。落款时间为：2008年8月20日。</p>
记者注意到，落款为2008年8月20日的“呈请发还物品报告书”中写道：“……2008年7月17日北京莫少平律师事务所受方宏进委托退还赃款100万元并赔偿今麦郎食品有限公司损失20万元，现根据河北省公安机关办理经济案件程序106条法规，将100万和赔偿损失的20万元，共计120万发还给今麦郎食品有限公司。”</p>
关于上述的“106条法规”，河北省公安厅官网查询到的内容为：“凡是发还给被害人的物品、文件，应当制作《呈请发还物品报告书》，经县级以上公安机关负责人批准后，开具《发还物品、文件清单》一式两份，一份附卷，一份交给领取人。”执行统筹/朱顺忠</p>

继李嘉欣、尹恩惠、郭碧婷、张雨绮、周韦彤、颖儿、刘芸之后，“紫霞仙子”朱茵也将加盟东方卫视明星时尚真人秀《女神新装》，尽管节目本身带着一定的比赛性质，和其他的女明星一起PK,朱茵心态却很轻松，“就算是比赛我也不会当做比赛，这样我就没压力了”。
第一次来到《女神新装》，朱茵连称收获很大，“我觉得这个挺好玩啊，可以和设计师合作一下，从鞋子到手镯，在不同的设计师身上可以学到不同的风格”。私下朱茵也接触过服装设计，“我上学的时候有一节课是教怎么裁剪衣服。女生要学的东西，比如煮饭、女红，我都懂。”对于比赛的输赢，朱茵也很看得开，“不用把它当做是比赛，在我心里面觉得既然我们有缘分，大家有机会碰面，有一些火花也是好事来的，没有压力出来的东西更自然”。她更表示，即便自己设计的衣服不被其他人看好，也不会介怀，“因为坦白说，你看见一件衣服，你觉得好看就说好看，你觉得不好看那要看你是怎么样的说法，我既然来了就不会介意这些”。</p>
因为在《大话西游》中出演紫霞仙子，颜值爆表、俏皮动人的朱茵成为无数粉丝心中的女神，并一度被评为“最让男人心动的女人”。在被问及怎么看“女神”这个称呼时，朱茵说：“如果女神一直在神坛，不把自己的姿态放低一点，那有用吗？我的意思是要了解大家的心态，比如你演戏如果你没有揣摩到别人的心态，那你怎么演呢？因为是女神，就更加要了解所有人的感觉，我是这样理解的，反正我就是这样的一个人。”那朱茵心中的女神又是谁？朱茵想都不想，说出了陈法蓉，“她大方得体，绝对是香港小姐的代表人物”。接下来《女神新装》还有闺蜜主题，是否会邀请陈法蓉前来呢？朱茵说：“要看她的档期，我是想邀请的，因为她是我非常要好非常爱的人，所以我希望她能来陪我。”新快报记者 曾乐
中新网9月9日电 据台湾“中国时报”报道，近日，歌手彭佳慧接受采访时透露，自己近20年来中午饭几乎只吃榨菜肉丝面，单靠着省吃俭用买房置业，至今手上已有3间房。</p>
近日，彭佳慧表示，自己当初打拼，身上存款只有7元。但她秉持着“省小钱，花大钱”的信念，把钱花在大额房贷上，出道第2年就存钱买房置业，多年下来从小房子换称大房子，目前已拥有3间房产的小富婆。</p>
彭佳慧表示，她平时吃饭没有“一定要选餐厅”的观念，只要没有饭局或通告，她每天的午餐，一定是榨菜肉丝面，“来台北近20年没有改变”。而靠着省钱买房当存款，彭佳慧现在已有3处房产，未来她最大的愿望，就是回到故乡，买一块地，盖一间自己喜欢的房子。</p>
中新网9月10日电 据香港“明报”消息，去年，柯震东因涉毒被捕，演艺事业就此停摆。近日，经历吸毒风波后，他首次接代言工作，为电玩游戏做宣传，还在广告中扮演郭靖。他表示，很久没有演戏，有戏瘾想复出拍戏。</p>
谈及近日因《我的少女时代》走红的好友王大陆，被问是否感到有威胁，柯震东笑说：“先给他接管一下，我改天再回来！”此外，他又给对方忠告，说人红了不要太嚣张，不然会像他摔得鼻青脸肿。</p>
9月8日凌晨，杨丞琳在微博写下一段心灵鸡汤：“我觉得，什么事都要刚刚好。刚刚好，即是最好！”此后，杨臣刚转发并调侃道：“谢谢妹妹！”</p>
对此，网友们纷纷调侃：“这脸能防弹！”、“刚刚的脸不见了”、“杠杠真机智！！！是在下输了！！！”、“刚刚可以靠才华，却偏偏选择了逗比。”。</p>
在昨日进行的2018年世界杯预选赛亚洲区小组赛中，面对弱旅马尔代夫，中国队以3比0战胜对手，顺利收获三分，也取得了本次世预赛小组阶段的第二场胜利。</p>
赛前压力大</p>
今年4月，当世界杯亚洲区预选赛分组揭晓后，所有人都认为中国队抽到了“上上签”，然而两场比赛之后，一胜一平的国足仅仅位列第三，排在中国香港和卡塔尔之后。</p>
对阵马尔代夫的比赛，国足虽然大获全胜，但在运动战中并没有创造出多少机会，尤其上半场时进攻脱节，依旧找不到破密集防守的办法。
对马尔代夫赛前，沈阳当地的《沈阳地铁报》用一整版为国足鼓劲：“中国男足今晚在福地沈阳若赢不了马尔代夫队，我就把这个头版吃了！”署名是“贼拉热爱中国足球的沈阳球迷”。</p>
赛场条件差</p>
在本场比赛中，场地虽然看起来绿油油，但整体并不平坦。比赛中，中国队的短传质量也多少受到了草皮的影响。</p>
本场比赛双方踢得非常别扭，场上的草皮多次影响球员的技战术动作，首粒进球就是伊姆兰出击和阿姆德罕相撞，给了于大宝倒地铲射的机会，整个过程草皮横飞，十分“有趣”。此后中国队又是连续被草皮所困扰，于汉超的突破传球与郑龙的怒射都受到草皮不佳的干扰。</p>
更有趣的是此后郑龙获得一次定位球机会，在发球前，郑龙抓起地上一块脱落的草皮，然后小心地将其摁在远处，才敢主罚任意球。比赛30分钟后，经过不断踩踏的草皮已是坑坑洼洼的乱七八糟，看到这一切后，有球迷调侃道：“这样的场地和中国队的水平相得益彰。”还有球迷则直言：“中国男足的脚下技术和比赛场地哪个更烂？”</p>
演唱会添乱</p>
4天后，周杰伦的演唱会就要在沈阳奥体中心举行，由于舞台搭建问题，球场草皮的状况相当糟糕。不仅如此，一侧球门后方还搭起了金属舞台，占去了大片球迷区域。</p>
有媒体发表微博称周杰伦给国足添堵，这样搭建舞台两队比赛都受影响，在世预赛这么重要的比赛中实属罕见。这条微博引发了周杰伦歌迷的不满，他们回应称：周杰伦演唱会已于去年通过审批，今年7月13日开始公开售票，不存在周杰伦演唱会干扰国足比赛的说法。其次，国足比赛场地是在8月临时更改至沈阳的，周杰伦演唱会合同在先。</p>
在9月6日的训练中，佩兰面对糟糕的场地条件频频摇头，但队长郑智表示，“作为球员在比赛时应该更专注，不管场地有什么东西，我们只管把球踢进球门。”</p>
对手很顽强</p>
对于中国队的对手马尔代夫，很多球迷还记得2001年的那场10比1。</p>
目前马尔代夫世界排名在150名开外，整体以业余球员为主。而据马尔代夫媒体赛前透露，队中唯一的职业球员，主力前锋阿里・阿什法克因伤并未随队参战。</p>
不过在本次世预赛小组赛中，马尔代夫表现顽强，前两战仅仅是以0比1和0比2不敌卡塔尔和中国香港。</p>
本场比赛，马尔代夫足协为了激励球员，历史性地开出了赢球奖金。只要球队赢球，每人将拿到10000拉菲亚的奖金(约合人民币4125元)。</p>
本场获胜后，中国队下场世界杯预选赛将于10月8日客场迎战卡塔尔。辛倜
